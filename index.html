배경

구현

  • Ganache 기반 로컬 블록체인 테스트 환경 구성
  • IPFS/Pinata를 활용한 NFT 이미지 및 메타데이터 저장 흐름 구성
  • Express.js를 중개 서버로 두어 클라이언트 요청과 블록체인 네트워크 상호작용 분리

학부 연구생 · 사이드 채널 분석 프로젝트
2021

GitHub: github.com/poketopa/side-channel-attack-cpa-on-aes-128

배경

AES-128 암호화 과정에서 수집한 전력 트레이스를 기반으로 상관 전력 분석(CPA)을 수행해 AES 키를 복원하는 연구 프로젝트입니다.

구현
  • 오실로스코프와 Arduino 기반 AES 암호화 환경에서 전력 트레이스 수집
  • estraces를 이용해 트레이스 데이터를 변환하고 평문·암호문·전력 트레이스 데이터 정리
  • C로 CPA 알고리즘을 구현하고 MATLAB으로 트레이스 및 상관계수 시각화
  • 추출한 10라운드 키를 기반으로 Python 스크립트를 통해 AES 마스터 키 역연산 복원
결과
  • CPA를 통해 AES 10라운드 키를 추출하고 키 스케줄 역연산으로 마스터 키를 복원
  • AES 구현, 마스킹 구현, 전력 분석 실험 흐름을 함께 정리
